# Upload file with Multer

## Detail

ตัวอย่างเว็บอัปโหลดไฟล์ด้วย Express มีทั้งอัปโหลดทีละไฟล์หรือหลายไฟล์ในครั้งเดียว

## Screenshort

![demo](/images/demo.png)

## Usage

```bash
    npm install
    # dev
    npm run dev
    # build
    npm run build
    npx ts-node dist/app.js
```
